Yeoman 快速构建新项目目录结构脚手架工具

发表于 2016-07-12   |   分类于 工程化   |     |   阅读次数

因为工作中经常需要创建新的项目目录（不要问我为什么），每次开始新的项目都需要新建一次前端目录结构、流程化构建等等，后来发现使用Yeoman新建别工程目录很方便，就想自己也定制一套，慢慢完善。

安装配置

  • 全局安装yo和yeoman-generator： npm install -g yo ， npm install --save yeoman-generator
  • 创建一个以 generator- 开头的文件夹，例如： generator-name
  • 在命令行运行 npm init 来构建package.json文件，其中 name 属性必须以 generator- 前缀开头。

React-Native基于windows的android环境搭建

发表于 2016-06-27   |   分类于 react   |     |   阅读次数

首先要说一下React-Native对windos环境配置的问题，用mac按照教程从开始到结束只要安装完，就可以无错运行起来。windows各种报错，研究了2天还是很多错误不知道为什么。就在要放弃治疗的时候，他成功运行起来了。

注意点一 ：翻墙

下载 Android Studio、Android SDK、npm install -g react-native-cli、Genymotion（安卓模拟器）都需要翻墙，不然速度非常慢，当下载失败的时候就会报错。

解决办法：

sdk可以使用国内镜像：http://www.androiddevtools.cn/ （推荐使用腾讯的镜像，如果加速不明显，可以尝试重启电脑）

npm 可以使用淘宝的cnpm镜像安装 react-native-cli。
